# 🐴 2026馬年運勢測算

一個簡單、現代、好用的馬年運勢網頁應用，結合易經和生日書為你預測2026年的運勢。

## ✨ 功能特色

- 🎨 **現代簡約設計** - 清爽的漸變色介面，使用體驗流暢
- 📅 **生日測算** - 只需輸入生日即可獲得專屬運勢
- ☯️ **易經卦象** - 根據生日計算對應的易經卦象
- 📖 **生日書元素** - 結合五行和月份特質分析
- 🎯 **四大運勢版塊**：
  - 新年祝福
  - 鼓勵的話
  - 需要注意的事
  - 易經卦象解析
- 📚 **適合所有人** - 使用CEFR B1程度的中文，簡單易懂
- 💡 **拼音輔助** - 難詞附有拼音和簡單說明（滑鼠移到帶底線的詞語上即可查看）

## 🚀 使用方法

### 線上使用
直接訪問：**https://newwineteach.github.io/20260101/**

### 本地使用
1. 下載 `index.html` 文件
2. 在瀏覽器中打開
3. 選擇你的生日
4. 點擊「查看我的2026運勢」按鈕
5. 查看你的專屬馬年運勢！
